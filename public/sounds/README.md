# 声音文件

这个文件夹包含应用程序使用的所有声音效果。

## 声音文件列表

- click.mp3 - 点击按钮声音
- success.mp3 - 成功操作声音
- failure.mp3 - 失败操作声音
- game-start.mp3 - 游戏开始声音
- game-end.mp3 - 游戏结束声音
- card-flip.mp3 - 卡片翻动声音
- collect.mp3 - 收集物品声音
- match.mp3 - 匹配成功声音
- level-up.mp3 - 升级声音
- drag.mp3 - 拖动声音
- drop.mp3 - 放置声音
- achievement.mp3 - 成就解锁声音

## 注意

请确保所有声音文件都是免版权的，或者您拥有使用它们的权利。
